## 🚀 What is this repository?

This is not just a collection of solved problems.

This repository is a **daily log of how I improve my problem-solving mindset** using Data Structures and Algorithms in Python.

Each problem represents:

* A mistake I learned from
* A pattern I discovered
* A better way to think

---

## ⚡ My System (Not Just Practice)

Instead of randomly solving problems, I follow a system:

### 🔁 1. Understand

* Break the problem into smaller parts
* Identify patterns (sliding window, two pointers, etc.)

### 🧩 2. Solve

* First attempt → brute force
* Second attempt → optimized

### 🔍 3. Reflect

* What did I miss?
* Can this be reused in another problem?

### 🧠 4. Store the Pattern

Every problem becomes part of a mental library.
